## 📋 Описание проекта
Разработан RAG-чатбот для ответов на вопросы по книге «*Python. Исчерпывающее руководство*» (368 страниц). Система использует **векторный поиск по смыслу** (не по ключевым словам) для извлечения релевантных фрагментов текста и генерации точных ответов на русском языке через **DeepSeek API**. 

Пользователь может задавать вопросы через **веб-интерфейс** и получать **ответы с опорой на контекст книги**. Если же вопрос не касается содержания книги, то модель это сообщает и игнорирует его.

### 📚 Библиотеки
* langchain, langchain-community, langchain-openai
* chromadb
* sentence-transformers
* gradio
* pypdf2, pdfplumber
* python-dotenv
* torch
* pydantic

### ⚙️ Используемые модели
* `sentence-transformers/paraphrase-multilingual-MiniLM-L12-v2` - векторизация текста
* `deepseek-chat` - генерация ответов на вопросы. LLM (API)
