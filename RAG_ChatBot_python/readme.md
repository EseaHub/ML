## 📋 Описание проекта
Разработан RAG-чатбот для ответов на вопросы по книге «Python. Исчерпывающее руководство» (368 страниц). Система использует векторный поиск по смыслу (не по ключевым словам) для извлечения релевантных фрагментов текста и генерации точных ответов на русском языке через DeepSeek API. 

Пользователь может задавать вопросы через веб-интерфейс и получать ответы с опорой на контекст книги. Если же вопрос не затраг

### 📚 Библиотеки
* langchain, langchain-community, langchain-openai
* chromadb
* sentence-transformers
* gradio
* pypdf2, pdfplumber
* python-dotenv
* torch
* pydantic

### ⚙️ Используемые модели
* `sentence-transformers/paraphrase-multilingual-MiniLM-L12-v2` - векторизация текста
* `deepseek-chat` - генерация ответов на вопросы. LLM (API)


### Результаты
* Проведена <ins>обработка данных</ins>: **368 страниц** представили **1396 семантических чанков**.

* <ins>Функция очистки чанков</ins> помогла избавиться от мусорных символов, сократя их общее количество с **1396 чанков** до **1389 штук**.

* <ins>Поиск</ins>: косинусное сходство через multilingual embeddings (k=3)

* <ins>Скорость ответа</ins>: **2-5 секунд** (включая API-запрос к DeepSeek)

* <ins>Точность</ins>: контекстный поиск по смыслу с инструкцией модели не выдумывать ответы
