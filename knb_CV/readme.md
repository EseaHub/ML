## Игра «Камень-Ножницы-Бумага» с управлением жестами
Групповой проект, в котором я выступил в роли TeamLead’a: проявил инициативу по распределению задач в команде, контролю выполнения, защите проекта.

Целью было разработать приложение для игры в «Камень-Ножницы-Бумага» через веб-камеру.

## Средства разработки
* Реализовано распознавание жестов рук в реальном времени с помощью MediaPipe и OpenCV
* Интегрировали библиотеку PIL для отрисовки текста, обеспечив интерфейс с поддержкой кириллицы
* Стек: Python, OpenCV, MediaPipe, Pillow, NumPy
