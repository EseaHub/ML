## pyTorch (Dogs vs Snakes)
📄 Свёрточная нейросеть, которая классифицирует картинки из датасета на "изображение собаки" и "изображение змеи".

🎯 Accuracy = **88%**.

📚 Библиотеки:
* torch
* numpy
* sklearn
* matplotlib


## tf_CNN (MNIST, Cats_vs_Dogs, VGG19)
### 🔢 MNIST
Написание мультиклассового классификатора на определение рукописных цифр.

Accuracy = **99%**.

### 🖼️ Cats_vs_Dogs
Создание бинарного классификатора изображений кошек и собак из набора данных Cats vs. Dogs.

Accuracy = **87%**.

### 🧠 VGG19
Использование предварительно обученной на наборе ImageNet нейронной сети VGG19 для увеличения точности классификации изображений из датасета Cats vs. Dogs.

Accuracy = **98%**.

📚 Библиотеки:
* tensorflow\keras
* numpy
* sklearn
* matplotlib, seaborn
