### 📋 Цель проекта
У нас есть таблица excel с металлургического завода Русал за сентябрь 2022 года. В ней по дням расписаны показатели основных примесей в ваннах, в которых варят алюминий. Нашей задачей является построить модель, которая будет по замерам, сделанными до варки, прогнозировать итоговую марку продукта. При этом перебить точность ручного замера, которая составляет 55%.

### 📚 Библиотеки
* numpy
* pandas
* matplotlib, seaborn
* sklearn
* CatBoost, LightGBM, XGBoost
